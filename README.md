# English AI Coach

Flalingo raporları, NotebookLM export, haftalık takip, XP, quiz ve AI speaking için Next.js demo proje.

## Çalıştırma

```bash
npm install
npm run dev
```

## Vercel Environment Variables

- `GEMINI_API_KEY`
- `XAI_API_KEY`

Bu demo frontend tarafında çalışır. API route'ları şimdilik güvenli stub olarak bırakılmıştır.
